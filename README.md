# IronPulse Fitness - Gym Website

A modern, responsive single-page gym website showcasing facilities, services, trainers, and membership plans.

## Features

- **Responsive Design**: Works on desktop, tablet, and mobile
- **Smooth Scrolling**: Navigation with active state highlighting
- **Interactive Tabs**: Services section with tab switching
- **Pricing Toggle**: Monthly/Yearly billing switch with animation
- **Animated Counters**: Statistics animation on scroll
- **Mobile Menu**: Hamburger navigation for mobile devices
- **Contact Forms**: Functional contact and newsletter forms
- **Back to Top**: Floating button appears on scroll

## Technologies

- HTML5
- CSS3 (Flexbox & Grid)
- Vanilla JavaScript
- Font Awesome Icons

## Sections

1. **Hero**: Full-screen banner with call-to-action
2. **About**: Gym story with statistics
3. **Facilities**: 6 facility cards with hover effects
4. **Services**: Tabbed interface for 4 services
5. **Trainers**: 4 trainer profiles
6. **Membership**: 3-tier pricing with toggle
7. **Contact**: Contact info and form
8. **Footer**: Links and newsletter

## Setup

1. Download all files
2. Open `index.html` in browser
3. No build process or dependencies required
